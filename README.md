# SistemaInventario3

## Descripción
Sistema web desarrollado en Java utilizando Servlets y JSP que permite gestionar un inventario de productos.

## Problema identificado
Las empresas requieren un sistema eficiente para controlar su inventario, ya que hacerlo manualmente genera errores y pérdida de información.

## Solución
Se desarrolló un sistema web que permite agregar y visualizar productos de forma sencilla y organizada.

## Arquitectura
El sistema está basado en el modelo MVC:
- Modelo: Producto.java
- Vista: JSP
- Controlador: Servlet

## Requerimientos
- Java JDK 11
- Apache Tomcat 10
- NetBeans
- Navegador web

## Instalación
1. Clonar repositorio
2. Abrir en NetBeans
3. Ejecutar con Tomcat

## Uso
- Agregar productos
- Visualizar inventario

## Contribución
1. Clonar repositorio
2. Crear rama
3. Hacer cambios
4. Subir cambios
